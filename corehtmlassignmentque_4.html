HTML tags

html tags are keywords that have opening point and closing point and are use to give structure and format to the web page that will be rendered on the browser.

Different types of html tags are =

  1. Heading tag:
    1. heading tag is use to render headings on te web page.
    2. we have mainly 6 heading elements h1 to h6=
      1. h1: most important hrading.
      2. h2
      3. h3
      4. h4
      5. h5
      6. h6: least important heading.
    3. dont use heading tag just to show the text bigger or bold.
  2. p tag :
    1. p tag or element stands for paragraph tag that is use rendered paragraph onthe webpage.
    2. for the p tag the browser itself add some margin before and after the paragraph on the webpage.
    3. paragraph always start form new line.
  3. br tag:
    1. br tag stand for break line tag.
    2. br tag break the current line starts rendering the remaining text from the next line.
    3. br tag has to separate closing tag.
  4. grouping tag or wrapper tag
    1. we have div as grouping tag .
    2. the grouping tag is use to make work more easier as it group the diferent tags in itself like paragraph tag so that the whole content in div can be customisable at ones.
    3. grouping tag has both opening and closing tag.
  5. formatting tag:
    1. formatting tags are used to structure the content on the web page so that important content will be rendered to the user effectively.
    2. There are different types of formatting tag:
      1. b: b tag is used to show the content in bold(just to make it more appealing).
      2. strong : strong tag is used to show the important content in bold(logical).
      3. i: i tag is used to show the content in italiced form
      4. em: em tag is used to show important content in italiced form.
      5. mark: mark tag is used to rendered the content in highlighted form.
      6. sub: this tag is used to render the content in subscripted form i.e a bit lower than the normal kine.
      7. sup:this keyword is ued to render the content in superscripted form i.e. a bit upper than the nirmal line.
      8. small: this tag is used to render the content in a 1 lower size than the base size.
      9. big: this tag is use to render the content in 1 size bigger than the base size.
      10. strike:this tag is used to render the deleted content with a strike through.
  6. a tag:
    1. a tag stand for anchor tag.
    2. this tag is use to render the hyperlinks in the webpage.
    3. there are many attribures of anchor tag available like format, hyref,etc.
