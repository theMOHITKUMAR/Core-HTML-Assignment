Web development refers to creation, building, and maintaince of web applications.
Web development is divided into two sub development

  1. Frontend development
  2. Backend development

Frontend development: part of the website that is visible to the user or we can say the paart of website with which the user can interact.

Backend development:the part of the website that is not visible to the user.
It is the server side of the website which is used to process, store and arrange the data.
